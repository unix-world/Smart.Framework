Change Account Password


[###ID|html###]
[%%%IF:LEN-KEYS:>0;%%%] [%%%IF:CRR-ID:==###ID###;%%%]
Important:
Your account already have a private key set.

Before you change your password be sure to make a safe backup of your private key which cannot be recovered if any fatal error will happen during this operation. If the private key is lost some sensitive encrypted information can no more be decrypted and will become inaccesible forever.
[%%%ELSE:CRR-ID%%%]
Important:
The selected account already have a private key set.

Since you are about to change the password for a different account than yours be sure to ask the account owner first to have a safe backup of it's own private key. If any fatal error will happen during this operation the private key may be lost and will can not be recovered anymore without a backup. If the private key is lost some sensitive encrypted information can no more be decrypted and will become inaccesible forever.
